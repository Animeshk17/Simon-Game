# Simon-Game
An implementation of the popular 80s game Simon using HTML, CSS and JavaScript. 
\
The game is simple. Repeat the pattern shown in the exact order. At each level a new color
will be added to the pattern and you have to remember the pattern from the beginning to the end and click the buttons in the same order. One mistake and it's GAME OVER.
\
